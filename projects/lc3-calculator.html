lc3-calculator

LC-3 Assembly Language Calculator

Simple calculator built using the LC-3 assembly language. Has the ability to parse strings with nested parentheses, and supports the four basic operations ( + - * /)

Usage

Enter a string of operations, as one would with a scientific calculator, then press enter. The program will then solve it and output the result. Operations are performed in the standard PEMDAS order (although exponents are not supported yet!). Operations within parentheses are evaluated first, then multiplication and division is performed at equal precedence, in the order of appearance from left to right. Then the same happens with addition.
Ex:
7*(6-(4+5))/3
7*(6-9)/3
7*(-3)/3
-21/3
-7

Limitations

  • The LC-3 replicates a 16-bit computer, so integers are only 16-bit. Values can range from -32,768 to 32,767.
  • All values are integers, and floats are not supported. Most noticeable with division. Ex: 5/2 = 2, not 2.5 as one would expect.
